62 /// Class for an Attribute of a templated type.
63 /**
64  TypeAttribute is the templated class that derives from an Attribute and completes the values (current and initial) and type for a Nomad parameter. A specific type of attribute is obtained by calling the templated AttributeFactory::Create function.
65  */
