typedef std::function<SuccessType(const EvalPointPtr &p1, const EvalPointPtr &p2, const Double& hMax)> ComputeSuccessFunction

Definition for compute success type function.

A function of this type compares two EvalPoints, and returns the SuccessType resulting from the comparison. The function is a member of ComputeSuccessType class and set using ComputeSuccessType::setComputeSuccessTypeFunction.
For example, computing success type is changed when optimizing a model instead of blackbox.
