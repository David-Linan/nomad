◆ getParentOfType()

template<typename T >
T Step::getParentOfType ( const bool  stopAtAlgo = true) const
inline

Template function to get the parent of given type.

Starting with parent of current Step, and going through ancestors, get first Step that is of type T. By default, stop if an Algorithm is found. Returned Step could be irrelevant otherwise. To go further up than an Algorithm, set optional parameter stopAtAlgo to false.
