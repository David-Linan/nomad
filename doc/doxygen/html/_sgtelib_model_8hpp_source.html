59 /// Class for implementation algorithms using Bastien Talgorn's sgtelib.
60 /**
61  * When used as an algorithm by itself:
62  * 1- Best points (with respect to blackbox evaluation) in the cache are found.
63  * - If the cache is empty, X0 points are used.
64  * 2- These points are used to build a dynamic model.
65  * 3- The model is optimized. This gives oracle points.
66  * 4- The oracle points are evaluated by the blackbox.
67  * 5- As long as new oracle points are found, the process is repeated.
68  *
69  * When used by Mads SearchMethod SgtelibSearchMethod:
70  * - Steps 1, 2 and 3 are the same (X0 is never used: always the cache).
71  * - The oracle points are send back to SgtelibSearchMethod, which takes care
72  * of projecting them to mesh and evaluate them.
73  *
74  * Reference: File Sgtelib_Model_Manager.cpp in NOMAD 3.9.1
75  * Author: Bastien Talgorn
76  */
