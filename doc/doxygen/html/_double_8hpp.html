std::istream& operator>> ( std::istream &  in,
Double &  d 
)

Input.

  • Allows the input of double objects with operator >>.
  • Can read undefined values (parameter UNDEF_STR with default "-".)
  • Example:
    Double d1 , d2;
    std::cout << "Enter d1 and d2: ";
    std::cin >> d1 >> d2;
    std::cout << "d1 and d2 are equal to " << d1 << " and " << d2 << std::endl;
