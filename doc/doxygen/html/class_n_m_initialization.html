Detailed Description

Class for Nelder Mead initialization.

For Step 0 of NM algorithm: Create and evaluate (if needed) trial points that will be used to form simplex. The points are put in cache.

◆ startImp()

virtual void NMInitialization::startImp ( )
overrideprivatevirtual

Implementation of start task.

If needed, generate trial points and put them in cache to form simplex. For a standalone optimization (NM_OPTIMIZATION true), initial trial points must be generated to form a valid simplex around x0. Otherwise, the cache will be used to construct the simplex.
