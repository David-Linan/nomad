Detailed Description

Final class to generate points for single pass on a given frame center. Used as Search method. The start, run and end tasks are empty. No evaluations are performed. The QuadModelSinglePass::generateTrialPoints function manages the creation process. The sample set to build the quad model is created by calling QuadModelIteration::startImp(). The points are not projected on mesh (done in SearchMethodBase).
