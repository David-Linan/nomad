Detailed Description

Class for the evaluator.

Evaluation of a point can be done by calling an external executable (provided in BB_EXE parameter) or by redefining the evaluation function Evaluator::eval_x(). /n To evaluate a block of points, the user must redefine Evaluator::eval_block() or make sure the external executable can evaluate all the provided points. /n
