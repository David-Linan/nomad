Detailed Description

Abstract base class for cache.

Note
Most methods of this are pure virtual and must be implemented in a derived class.

The cache itself is implemented in derived classes. It could be a set (CacheSet), an unordered_set (CacheSet with precompiler option USE_UNORDEREDSET), map, multimap, SQL database, etc.

virtual bool CacheBase::smartInsert ( const EvalPoint &  evalPoint,
const short  maxNumberEval = 1,
const EvalType &  evalType = EvalType::BB 
)
pure virtual

Insert evalPoint in cache.

evalPoint's tag (mutable) is updated. If insertion worked, the point was not in the cache before. Return true. If insertion did not work, the point was in the cache before. _nbCacheHits is incremented. Depending on its EvalStatus, on maxNumberEval, and on the evalType, return true if it should be evaluated again, false otherwise.

Parameters
evalPointThe eval point to insert in the cache – IN.
maxNumberEvalThe max number of evals – IN.
evalTypeLook at the Blackbox or Model eval of the EvalPoint – IN.
Returns
A boolean indicating if we should eval this point.
