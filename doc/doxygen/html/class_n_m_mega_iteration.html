Detailed Description

Class for Nelder Mead mega iteration.

Manager for Nelder Mead iterations starts, runs and ends. Steps:

  • Generate points over simplices (start).
  • Evaluate points (run)
  • Post-processing (end)

Implementation of run task.

The algorithm iterations are started, ran and ended sequentially until a stop reason to terminate is obtained.
We have a success if either a better xFeas or a dominating or partial success for xInf was found. See Algorithm 12.2 from DFBO.

Implements MegaIteration.

Implementation of start task.

Create a Nelder Mead iteration for a current incumbent: use xFeas or xInf from the barrier.

Note
Running the algorithm requires a single iteration object with several start, run, end for the various iterations of the algorithm. This allows to easily maintain a proper simplex in a single NMIteration object.

Implements MegaIteration.
