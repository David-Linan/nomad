54 /// Class to create trial points by performing quadratic model optimization using Mads
55 /**
56  - Start, run and end tasks are performed.
57  - Start: the quadratic model optimization problem is setup and solved by calling startImp. Call ::generateTrialPoints.
58  - Run: trial (oracle) points are evaluated with EvalType::BB. Set the stop reason.
59  - End: Remove from cache EvalType::MODEL only cache points.
60  */

100  /// Generate new points to evaluate
101  /**
102  - Setup the evaluator control parameters.
103  - Manage display of sub-optimization.
104  - Setup evaluator (EvalType::MODEL) and success type identification function.
105  - Setup the bounds and fixed variables from the trainingSet of the quadratic model.
106  - Setup run and pb parameters for Mads
107  - Perform start, run and end tasks on Mads.
