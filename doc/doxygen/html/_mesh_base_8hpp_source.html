66 /// The generic class for meshes (discretization of design variables space).
67 /**
68  \note To be implemented by the derived mesh used by an algorithm.
69 
70  \note This class encompasses the mesh discretization and the frame on it.
71 
72  * In the algorithm, we have delta for mesh size parameter and
73  Delta for frame size parameter. To avoid confusion in function calls,
74  using deltaMeshSize and DeltaFrameSize in function names.
75  * Functions for delta: MeshBase::getdeltaMeshSize, MeshBase::updatedeltaMeshSize
76  * Functions for Delta: MeshBase::getDeltaFrameSize, MeshBase::enlargeDeltaFrameSize, MeshBase::refineDeltaFrameSize
77  */

79 {
