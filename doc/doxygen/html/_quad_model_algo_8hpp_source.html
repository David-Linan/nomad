55 /// Class for implementation of quadratic model optimization algorithm using Bastien Talgorn's sgtelib.
56 /**
57  * Use the start, run and end tasks. Iterate on the following sequence:
58  *
59  * 1- Points provided as X0s and points in cache are put in a training set.
60  * 2- These points are used to build a dynamic model.
61  * 3- The model is optimized. This gives oracle points.
62  * 4- The oracle points are evaluated by the blackbox.
63  * 5- As long as new oracle points are found, the process is repeated.
64  *
65  * When used by Mads SearchMethod (QuadSearchMethod):
66  * - Steps 1, 2, 3 and 4 are the same.
67  * - The oracle points are send back to QuadSearchMethod, which takes care
68  * of projecting them to mesh and evaluate them.
69  *
70  * Training set and model are stored here to allow access to other Quad classes.
71  *
72  */
73
