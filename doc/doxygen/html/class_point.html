Detailed Description

Class for the representation of a point.

A point is defined by its size and its coordinates from ArrayOfDouble. Addition and comparison (< and weak less) operators are provided. A point can be converted from a sub space to a full space and the reverse (Point::makeFullSpacePointFromFixed and Point::makeSubSpacePointFromFixed).
